Abertura de Chamados via Automação

Preencha os campos abaixo para configurar a abertura automática de chamados no ServiceNow. Os dados são enviados ao script Python que processa e registra cada chamado individualmente.

Como funciona: O analista informa os e-mails dos usuários e configura os parâmetros do chamado. O script Python utiliza esses dados para abrir um chamado no ServiceNow para cada usuário informado, de forma automática. Campos marcados com Auto são preenchidos automaticamente pelo sistema.

Campos marcados com asterisco são obrigatórios.
